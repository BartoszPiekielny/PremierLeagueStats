# Premier League Stats

ASP.NET Core Web API project for managing Premier League data.

## Technologies

* ASP.NET Core 8
* Entity Framework Core
* MySQL
* AutoMapper
* JWT Authentication
* ASP.NET Identity
* xUnit
* Docker
* DocFX
* football-data.org API

## Features

* Clubs management
* Players management
* Coaches management
* Stadiums management
* Import data from football-data.org
* Search players by:

  * name
  * nationality
  * club
  * position
* JWT authentication
* Admin/User roles
* Docker support
* Unit tests
